===============================================================================

Python library for generating charts for Amazon Web Services based on Amazon
CloudWatch data.


See REQUIREMENTS.txt for list of dependencies.


Install all dependencies
$ pip install -r REQUIREMENTS.txt


Install pyawschart with pip
$ pip install pyawschart
